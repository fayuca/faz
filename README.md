# faz

Simple finance tracker backend built with Spring Boot.

## Features

- Create transactions
- Update transactions
- Delete transactions
- List transactions
- Filter transactions by description
- Validation and error handling
- Unit, controller, and integration tests

## Stack

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- HSQLDB
- JUnit 5
- Mockito
- MockMvc
